## 角色定义

| 角色 | 说明 |
|------|------|
| 系统管理员 | 负责系统级配置、用户账号和权限管理 |
| 图书管理员 | 负责图书的入库、编目、借还操作、逾期处理 |
| 普通读者 | 已注册的读者，可检索、借阅、续借、查看借阅历史 |
| 游客 | 未注册用户，仅可浏览和检索图书信息 |

## 一、系统管理员

### US-ADM-01 用户账号管理

> 作为**系统管理员**，我想要**创建、编辑、启用和禁用用户账号**，以便**管理系统的合法访问者并保障系统安全**。

**验收标准：**
- [ ] 可创建新用户账号，必填字段：用户名、密码、角色（系统管理员/图书管理员/普通读者）
- [ ] 可编辑已有用户的基本信息（姓名、联系方式、角色）
- [ ] 可禁用用户账号，禁用后该用户无法登录
- [ ] 可重新启用已禁用的用户账号
- [ ] 用户列表支持按角色、状态筛选
- [ ] 用户列表支持按用户名搜索
- [ ] 删除用户为软删除，保留历史数据关联

### US-ADM-02 角色与权限管理

> 作为**系统管理员**，我想要**定义和管理角色权限**，以便**确保不同角色的用户只能访问其授权范围内的功能**。

**验收标准：**
- [ ] 系统预置三种角色：系统管理员、图书管理员、普通读者
- [ ] 每个角色的功能权限可独立配置（菜单访问、操作权限）
- [ ] 权限变更后，已登录用户在下次操作时生效
- [ ] 权限配置界面以角色为维度展示所有可配置权限项

### US-ADM-03 系统配置管理

> 作为**系统管理员**，我想要**配置系统运行参数**，以便**系统能按不同的业务规则运行**。

**验收标准：**
- [ ] 可配置借阅规则：最大借阅册数（默认5本）
- [ ] 可配置借阅期限：单次借阅天数（默认30天）
- [ ] 可配置续借规则：最大续借次数（默认1次）、每次续借天数（默认30天）
- [ ] 可配置逾期罚则：每日逾期费用（默认0.1元/天）
- [ ] 配置修改后即时生效，无需重启服务
- [ ] 系统配置页面展示当前生效的所有参数

### US-ADM-04 操作日志查看

> 作为**系统管理员**，我想要**查看系统操作日志**，以便**追溯关键操作记录和排查异常**。

**验收标准：**
- [ ] 日志记录关键操作：用户登录/注销、图书入库/下架、借书/还书/续借
- [ ] 每条日志包含：操作时间、操作人、操作类型、操作对象、IP地址
- [ ] 支持按时间范围筛选日志
- [ ] 支持按操作类型和操作人筛选日志
- [ ] 日志不可被删除或修改

## 二、图书管理员

### US-LIB-01 图书入库

> 作为**图书管理员**，我想要**将新书录入系统**，以便**读者能够检索和借阅这些图书**。

**验收标准：**
- [ ] 入库表单包含：ISBN、书名、作者、出版社、出版日期、分类、定价、馆藏数量
- [ ] ISBN 可自动检索填充图书信息（调用公开 ISBN 接口）
- [ ] 支持同一 ISBN 增加馆藏复本数量
- [ ] 入库成功后自动生成唯一图书编号（条码号）
- [ ] 入库成功后图书状态为"可借"
- [ ] 必填字段为空时给出明确提示

### US-LIB-02 图书编目

> 作为**图书管理员**，我想要**对图书进行分类编目**，以便**读者能按分类浏览和检索图书**。

**验收标准：**
- [ ] 支持为图书设置/修改分类（使用中图法分类）
- [ ] 支持添加图书标签（关键词），标签数量不限
- [ ] 支持为图书添加简介和封面图片
- [ ] 编目信息修改后即时生效
- [ ] 可按分类树浏览所有藏书

### US-LIB-03 图书下架

> 作为**图书管理员**，我想要**对破损或过时的图书进行下架处理**，以便**保持馆藏信息的准确性**。

**验收标准：**
- [ ] 下架操作需填写下架原因（破损/遗失/过时/其他）
- [ ] 下架后图书状态变为"已下架"，不可被借阅
- [ ] 若有读者正在借阅该书，提示先归还再下架
- [ ] 支持恢复已下架的图书
- [ ] 下架图书记录保留在系统中，可在查询时筛选

### US-LIB-04 借书操作

> 作为**图书管理员**，我想要**为读者办理借书登记**，以便**记录图书借出状态和借阅信息**。

**验收标准：**
- [ ] 借书流程：扫描/输入读者证号 → 确认读者身份 → 扫描/输入图书编号 → 确认借出
- [ ] 系统自动校验读者借阅数量是否已达上限，超限提示不可借
- [ ] 系统自动校验读者是否有逾期未还图书，有逾期不可再借
- [ ] 系统自动校验读者是否有未缴罚款，有欠款不可再借
- [ ] 系统自动校验图书状态是否为"可借"，非可借状态不可借出
- [ ] 借阅成功后自动计算应还日期（当前日期 + 借阅期限）
- [ ] 借阅成功后图书状态变为"已借出"
- [ ] 支持批量借书（同一读者一次借多本）
- [ ] 打印/生成借阅凭条（可选）

### US-LIB-05 还书操作

> 作为**图书管理员**，我想要**为读者办理还书登记**，以便**图书恢复可借状态并完成借阅记录闭环**。

**验收标准：**
- [ ] 还书流程：扫描/输入图书编号 → 显示借阅信息 → 确认归还
- [ ] 系统自动判断是否逾期：若逾期，计算逾期天数和罚款金额
- [ ] 还书成功后图书状态恢复为"可借"
- [ ] 还书成功后借阅记录标记为"已归还"
- [ ] 若有逾期罚款，记录待缴罚款并提示管理员
- [ ] 支持批量还书

### US-LIB-06 逾期管理

> 作为**图书管理员**，我想要**查看和处理逾期借阅记录**，以便**及时催还图书和处理罚款**。

**验收标准：**
- [ ] 逾期列表展示所有逾期未还的借阅记录
- [ ] 每条逾期记录展示：读者信息、图书信息、应还日期、逾期天数、应付罚款
- [ ] 支持按逾期天数排序
- [ ] 支持处理罚款缴纳：标记罚款为"已缴纳"
- [ ] 缴纳罚款后读者可恢复借阅权限
- [ ] 支持导出逾期报表

### US-LIB-07 读者管理

> 作为**图书管理员**，我想要**管理读者信息**，以便**维护准确的读者档案和处理读者服务请求**。

**验收标准：**
- [ ] 可为读者办理注册（本质是创建"普通读者"角色用户账号）
- [ ] 可查看和编辑读者基本信息（联系方式、地址等）
- [ ] 可查看读者的当前借阅记录和借阅历史
- [ ] 可查看读者的逾期记录和罚款记录
- [ ] 可为读者办理挂失（挂失后不可借阅）

## 三、普通读者

### US-RDR-01 图书检索

> 作为**普通读者**，我想要**通过多种方式检索图书**，以便**快速找到我需要的图书**。

**验收标准：**
- [ ] 支持按书名关键词搜索（模糊匹配）
- [ ] 支持按作者搜索
- [ ] 支持按 ISBN 精确搜索
- [ ] 支持按分类浏览图书
- [ ] 搜索结果展示：书名、作者、封面缩略图、状态（可借/已借出）、馆藏位置
- [ ] 搜索结果支持按出版日期、书名排序
- [ ] 支持组合条件筛选（分类 + 状态 + 关键词）
- [ ] 点击图书可查看详细信息（简介、标签、借阅状态、所在位置）

### US-RDR-02 借阅图书

> 作为**普通读者**，我想要**在图书馆现场借阅图书**，以便**将图书带出阅读**。

**验收标准：**
- [ ] 读者到馆后由图书管理员操作借书（见 US-LIB-04）
- [ ] 读者可自助查看当前借阅数量上限和已借数量
- [ ] 系统在借书时自动校验借阅资格
- [ ] 借阅成功后读者可在个人中心查看借阅记录

### US-RDR-03 续借图书

> 作为**普通读者**，我想要**在线续借即将到期的图书**，以便**延长阅读时间而无需到馆**。

**验收标准：**
- [ ] 读者可在"我的借阅"中看到即将到期的图书
- [ ] 续借按钮在到期前 N 天内可用（N 由系统配置，默认 7 天）
- [ ] 续借前系统校验：是否超过最大续借次数、是否有他人预约该书
- [ ] 续借成功后应还日期更新为"原应还日期 + 续借天数"
- [ ] 续借成功后显示新的应还日期
- [ ] 续借次数用完后不显示续借按钮
- [ ] 若有他人已预约该书，不可续借

### US-RDR-04 查看借阅历史

> 作为**普通读者**，我想要**查看我的借阅历史和统计数据**，以便**了解我的阅读情况和借阅习惯**。

**验收标准：**
- [ ] 借阅历史列表展示：书名、借出日期、归还日期、状态（借阅中/已归还/逾期）
- [ ] 当前借阅的图书单独分组展示（"正在借阅"）
- [ ] 支持按状态筛选借阅记录
- [ ] 显示借阅统计数据：总借阅次数、本月借阅次数、当前在借数量
- [ ] 逾期记录高亮显示，并展示已缴/未缴罚款金额

### US-RDR-05 个人中心

> 作为**普通读者**，我想要**管理我的个人信息和密码**，以便**保持账户信息的准确和安全**。

**验收标准：**
- [ ] 可查看和编辑个人基本信息（姓名、联系方式、邮箱）
- [ ] 可修改登录密码（需验证旧密码）
- [ ] 显示当前借阅概况（借阅数量、即将到期提醒、逾期提醒）
- [ ] 显示待缴罚款金额（若有）

### US-RDR-06 图书预约（扩展场景）

> 作为**普通读者**，我想要**预约已被借出的图书**，以便**在该书归还时能优先借阅**。

**验收标准：**
- [ ] 当图书状态为"已借出"时，显示"预约"按钮
- [ ] 每位读者同时最多预约 3 本图书
- [ ] 预约图书归还后，系统通知预约读者（预留通知位）
- [ ] 预约保留期为 N 天（N 由系统配置），超期自动取消预约
- [ ] 读者可取消自己的预约

## 四、游客

### US-GST-01 图书浏览与检索

> 作为**游客**，我想要**浏览和检索图书馆藏**，以便**了解图书馆有哪些图书**。

**验收标准：**
- [ ] 可在首页直接进行图书搜索（按书名、作者、ISBN）
- [ ] 可按分类逐级浏览图书
- [ ] 搜索和浏览结果展示：书名、作者、出版社、封面缩略图、状态
- [ ] 可查看图书详细信息（简介、分类、标签）
- [ ] 图书状态展示为"可借"或"已借出"（游客可了解是否可借）

### US-GST-02 注册引导

> 作为**游客**，我想要**注册为系统用户**，以便**享受借阅服务**。

**验收标准：**
- [ ] 首页和导航栏展示"注册"入口
- [ ] 注册页面包含：用户名、密码、确认密码、姓名、联系方式（手机号/邮箱）
- [ ] 注册时校验用户名唯一性
- [ ] 注册时密码需符合强度要求（至少8位，含字母和数字）
- [ ] 注册成功后自动分配"普通读者"角色
- [ ] 注册成功后跳转到登录页面

### US-GST-03 登录

> 作为**已注册用户**，我想要**登录系统**，以便**使用我的角色对应的全部功能**。

**验收标准：**
- [ ] 登录表单包含：用户名、密码
- [ ] 登录成功后根据角色跳转到对应首页（管理员/管理员面板，读者/图书检索页面）
- [ ] 登录失败给出明确错误提示（用户名不存在 / 密码错误 / 账号已禁用）
- [ ] 连续登录失败 5 次后账号临时锁定 15 分钟
- [ ] 支持"记住我"功能（7天内免登录）
- [ ] 系统管理员和图书管理员使用同一管理后台登录入口
