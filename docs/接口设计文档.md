## 1. API 设计规范

### 1.1 通用约定

| 项目 | 约定 |
|------|------|
| 协议 | HTTPS（生产）/ HTTP（开发） |
| 数据格式 | JSON (`Content-Type: application/json`) |
| 字符编码 | UTF-8 |
| 认证方式 | Bearer Token (JWT)，在 Authorization Header 中传递 |
| URL 风格 | RESTful，资源名复数，层级不超过 3 层 |
| 版本管理 | URL 前缀 `/api/v1/`，后续升级增加 `/api/v2/` |

### 1.2 统一响应格式

**成功响应**：
```json
{
  "code": 200,
  "message": "ok",
  "data": { ... }
}
```

**分页响应**：
```json
{
  "code": 200,
  "message": "ok",
  "data": {
    "list": [ ... ],
    "total": 150,
    "page": 1,
    "pageSize": 20
  }
}
```

**错误响应**：
```json
{
  "code": 400,
  "message": "错误描述",
  "data": null
}
```

### 1.3 HTTP 状态码

| 状态码 | 说明 | 使用场景 |
|--------|------|---------|
| 200 | 成功 | GET/PUT 成功 |
| 201 | 已创建 | POST 成功创建资源 |
| 204 | 无内容 | DELETE 成功 |
| 400 | 请求参数错误 | 参数校验失败 |
| 401 | 未认证 | Token 缺失/过期/无效 |
| 403 | 无权限 | 角色权限不足 |
| 404 | 资源不存在 | 查询/更新/删除目标不存在 |
| 409 | 冲突 | 业务规则冲突（如重复操作） |
| 422 | 实体校验失败 | 数据校验不通过 |
| 429 | 请求过于频繁 | 速率限制触发 |
| 500 | 服务器内部错误 | 未预期的运行时错误 |

### 1.4 错误码体系

| code | message 模板 | 说明 |
|------|-------------|------|
| 40001 | 参数校验失败: {field} | 请求参数不符合规则 |
| 40101 | 请先登录 | Token 缺失 |
| 40102 | 登录已过期，请重新登录 | Token 过期 |
| 40103 | 认证失败 | Token 无效 |
| 40301 | 无操作权限 | 角色权限不足 |
| 40401 | {resource}不存在 | 资源不存在 |
| 40901 | 用户名已被注册 | 注册用户名重复 |
| 40902 | 手机号已被注册 | 手机号重复 |
| 40903 | 该图书已被借出 | 并发借书冲突 |
| 40904 | 该图书已归还 | 重复还书 |
| 40905 | 续借次数已用完 | 续借次数达上限 |
| 40906 | 借阅数量已达上限 | 读者借阅数量超限 |
| 40907 | 存在逾期未还图书 | 读者有逾期图书 |
| 40908 | 存在未缴罚款 | 读者有逾期罚款未缴 |
| 40909 | 账号已被禁用 | 账号禁用无法登录 |
| 40910 | 账号已临时锁定 | 登录失败次数过多 |
| 40911 | 图书有副本正在借出 | 下架时有副本借出中 |
| 40912 | 该图书已被其他读者预约 | 续借时被预约 |
| 50001 | 服务器内部错误 | 通用服务器错误 |

## 2. API 端点详细设计

### 2.1 认证模块 `/api/auth`

#### POST `/api/auth/register` — 用户注册

```
认证要求: 否
权限要求: 无
```

**请求参数** (Body JSON)：

| 参数 | 类型 | 必填 | 校验规则 |
|------|------|------|---------|
| username | string | 是 | 3-50 字符，字母开头 |
| password | string | 是 | 8-32 位，含字母和数字 |
| name | string | 是 | 1-50 字符 |
| phone | string | 是 | 1 开头的 11 位数字 |
| email | string | 否 | 合法邮箱格式 |

**请求示例**：
```json
{
  "username": "zhangsan",
  "password": "abc12345",
  "name": "张三",
  "phone": "13800138000",
  "email": "zhangsan@example.com"
}
```

**成功响应** (201)：
```json
{
  "code": 201,
  "message": "注册成功",
  "data": {
    "userId": 2,
    "username": "zhangsan",
    "readerNo": "R-20260704-00001"
  }
}
```

**错误响应**：
```json
// 409 — 用户名重复
{ "code": 40901, "message": "用户名已被注册", "data": null }

// 409 — 手机号重复
{ "code": 40902, "message": "手机号已被注册", "data": null }
```

---

#### POST `/api/auth/login` — 用户登录

```
认证要求: 否
权限要求: 无
速率限制: 同一 IP 每分钟最多 10 次
```

**请求参数** (Body JSON)：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 登录用户名 |
| password | string | 是 | 密码明文 |
| rememberMe | boolean | 否 | 记住我（Token 7 天有效），默认 false |

**请求示例**：
```json
{
  "username": "admin",
  "password": "admin123456",
  "rememberMe": false
}
```

**成功响应** (200)：
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
      "id": 1,
      "username": "admin",
      "role": "admin",
      "name": "系统管理员"
    }
  }
}
```

**错误响应**：
```json
// 401 — 用户名或密码错误（统一提示）
{ "code": 401, "message": "用户名或密码错误", "data": null }

// 409 — 账号禁用
{ "code": 40909, "message": "账号已被禁用", "data": null }

// 409 — 账号锁定
{ "code": 40910, "message": "账号已临时锁定，请15分钟后重试", "data": null }
```

### 2.6 借阅管理 `/api/borrows`

#### POST `/api/borrows` — 借书

```
认证要求: 是
权限要求: librarian / admin
```

**请求参数** (Body JSON)：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| readerNo | string | 是 | 读者证号或手机号 |
| barcodes | string[] | 是 | 图书条码列表，最多 5 本 |
| borrowDays | integer | 否 | 借阅天数，默认取系统配置 |

**校验规则**：
1. 读者存在且状态正常（非冻结/挂失/禁用）
2. 读者无逾期未还图书
3. 读者无未缴罚款
4. 读者当前借阅数 + 本次借阅数 ≤ 借阅上限
5. 所有图书条码有效且状态为"可借"

**请求示例**：
```json
{
  "readerNo": "R-20260704-00001",
  "barcodes": ["BC-20260704-00001", "BC-20260704-00003"],
  "borrowDays": 30
}
```

**成功响应** (201)：
```json
{
  "code": 201,
  "message": "借书成功",
  "data": {
    "borrows": [
      {
        "id": 1,
        "barcode": "BC-20260704-00001",
        "bookTitle": "JavaScript高级程序设计",
        "dueDate": "2026-08-03"
      },
      {
        "id": 2,
        "barcode": "BC-20260704-00003",
        "bookTitle": "深入理解计算机系统",
        "dueDate": "2026-08-03"
      }
    ]
  }
}
```

**错误响应**：
```json
// 404 — 读者不存在
{ "code": 40401, "message": "读者不存在", "data": null }

// 409 — 借阅数量已达上限
{ "code": 40906, "message": "借阅数量已达上限（当前5/5）", "data": null }

// 409 — 存在逾期未还
```
